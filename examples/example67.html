Phase E3.3

Docked Grouping and Pivot Tool Panels

side bar가 overlay가 아니라 grid width를 실제로 줄이는 docked shell로 붙습니다. shell을 열면 상단 tab에서 Grouping과 Pivot panel을 오가며 모델을 바로 바꿀 수 있습니다.

  • Grouping panel에서 그룹 열과 집계를 바로 켤 수 있습니다.
  • Pivot panel에서 pivot column과 value aggregation을 설정할 수 있습니다.
  • 닫으면 panel 폭은 빠지고 slim toggle만 남습니다.

현재 범위

sideBar.panels = ['grouping', 'pivot']
docked shell + width reservation
group model / aggregations editor
pivot model / value aggregation editor
