Phase E3.3

Filters Tool Panel

header menu를 열지 않고도 우측 rail의 Filters panel에서 column별 필터를 바로 편집할 수 있습니다. 현재는 single-column quick filter surface를 sidebar로 올린 형태이고, text / number / date / set 모드는 기존 filter model과 같은 계약을 사용합니다.

  • Region은 text ↔ set mode 전환을 확인할 수 있습니다.
  • Name, Score, Created At도 같은 panel 안에서 편집됩니다.
  • apply / clear는 현재 선택 column filter만 바꾸고 나머지 filter model은 유지합니다.

현재 범위

sideBar.panels = ['filters']
column list + embedded filter editor
text/number/date 2-clause AND
set filter + search
filterModel 양방향 동기화
Filtered Columns 0
Visible IDs - 

            
Filters panel starts open by default.
