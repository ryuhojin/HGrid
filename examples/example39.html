Example 39 - Keyboard Only Flow

Mouse 없이 navigation/selection/editing 경로를 검증하는 a11y keyboard demo.

Keyboard shortcuts
- Arrow / PageUp / PageDown / Home / End: navigate active cell
- Shift + Arrow: extend range selection
- Ctrl/Cmd + A: select all cells
- Enter or F2: start editing active editable cell
- Tab / Shift+Tab (while editing): commit and move to next/prev editable cell
- Escape (while editing): cancel edit
