Phase E3.1.2

Body Cell Context Menu

데이터 셀에서 우클릭하면 body-scoped context menu가 열립니다. menu payload는 현재 cell, row, rowKey, value, selection snapshot까지 포함합니다.

  • 우클릭한 셀이 선택되어 있지 않으면 해당 셀을 single selection으로 승격합니다.
  • 이미 선택된 셀을 우클릭하면 기존 range는 유지하고 active cell만 target으로 맞춥니다.
  • 편집 중에는 body context menu를 열지 않습니다.

Payload 요약

context.kind === 'cell'
context.rowIndex / dataIndex / rowKey
context.row / value / selection
Last Kind none
Active Cell none
