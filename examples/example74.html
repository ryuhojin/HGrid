Advanced Filter Builder

Filters tool panel 안에서 Quick와 Builder를 전환하면서 cross-column advanced filter를 구성하는 예제입니다. Builder는 top-level AND / OR와 다중 rule을 지원하고, quick column filter와 함께 병행할 수 있습니다.

  • Builder: 서로 다른 컬럼을 조합해 OR/AND rule을 만듭니다.
  • Quick + Builder: 기존 column filter와 고급 builder를 동시에 유지합니다.
  • Snapshot: 현재 quick filter, advanced filter, visible ids를 같이 봅니다.

Builder Model

grid.setAdvancedFilterModel({
  operator: 'or',
  rules: [
    { columnId: 'name', condition: { kind: 'text', operator: 'contains', value: 'ta' } },
    { columnId: 'region', condition: { kind: 'text', operator: 'equals', value: 'APAC' } }
  ]
});
