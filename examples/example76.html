Nested Advanced Filter Builder

Filters tool panel의 Builder surface에서 nested group를 조합하는 예제입니다. top-level AND/OR 아래에 group을 중첩하고, group 내부에서 rule과 또 다른 group을 같이 쌓을 수 있습니다.

  • Nested groups: root 아래에 group을 추가하고 group 내부에 rule을 더 넣습니다.
  • Cross-column logic: 서로 다른 컬럼을 AND / OR로 묶어 한 번에 필터링합니다.
  • Snapshot: 현재 advanced filter model과 visible ids를 같이 확인합니다.

Nested Builder Model

grid.setAdvancedFilterModel({ operator: 'or', rules: [ { columnId: 'status', condition: { kind: 'text', operator: 'equals', value: 'review' } }, { kind: 'group', operator: 'and', rules: [ { columnId: 'region', condition: { kind: 'text', operator: 'equals', value: 'APAC' } }, { columnId: 'score', condition: { kind: 'number', operator: 'gte', value: 70 } } ] } ] });
