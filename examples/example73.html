Body Context Built-In Actions

body cell 우클릭 메뉴에서 copy / filter built-in action을 opt-in으로 노출하는 예제입니다. custom hook 없이도 셀 값 복사, 행 복사, 선택 영역 복사, 현재 값으로 필터, 해당 열 필터 해제를 바로 실행할 수 있습니다.

  • Copy cell: 현재 셀의 표시 문자열을 복사합니다.
  • Copy row: 현재 보이는 일반 컬럼 기준으로 한 행을 TSV로 복사합니다.
  • Copy selection: 현재 selection TSV를 복사합니다.
  • Filter by this value: 현재 셀 값으로 set filter를 바로 적용합니다.
  • Clear column filter: 현재 열 filter만 해제합니다.

옵션

contextMenu: {
  enabled: true,
  builtInActions: ['copyCell', 'copyRow', 'copySelection', 'filterByValue', 'clearColumnFilter']
}
