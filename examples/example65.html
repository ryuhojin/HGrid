Phase E3.3

Columns Tool Panel

우측 rail의 Columns 버튼에서 열 구성 panel을 열고, visible / hidden 상태와 left / right pin을 바로 바꿀 수 있습니다. panel은 현재 visible columns가 아니라 전체 column catalog를 기준으로 그려지므로, 숨긴 열도 다시 복구할 수 있습니다.

  • Status를 숨긴 뒤 다시 켜서 full catalog 복구 경로를 확인합니다.
  • Name을 left pin으로 옮겨 renderer order와 pin 상태가 같이 바뀌는지 봅니다.
  • tool panel은 grid overlay 위에 떠 있지만 DOM virtualization hot path는 건드리지 않습니다.

현재 범위

sideBar.enabled = true
panels = ['columns']
defaultPanel = 'columns'
visible / hidden toggle
pin left / right / none
Visible Columns -
Pinned Left - 

            
Columns panel starts open by default.
